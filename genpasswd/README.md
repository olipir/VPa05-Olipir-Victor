# GenPasswd

Консольный менеджер паролей на Python с шифрованием.

## Возможности

- Шифрование паролей **Fernet** (библиотека `cryptography`)
- Мастер-пароль с хешированием **SHA-256**
- Хранение данных в **SQLite3**
- Интерактивный режим (REPL)

## Установка

```bash
pip install cryptography
```

## Использование

```bash
python main.py
```

При первом запуске программа запросит создание мастер-пароля.

### Команды

| Команда | Описание |
|---|---|
| `add` | Добавить новую запись |
| `get <name>` | Получить пароль по названию |
| `list` | Список всех записей (название и логин) |
| `delete <name>` | Удалить запись по названию |
| `new` | Сменить мастер-пароль |
| `exit` / `quit` | Выход |

## Структура проекта

- `main.py` — точка входа, CLI и REPL
- `db.py` — работа с SQLite3 (таблицы `master`, `entries`)
- `crypto_utils.py` — генерация ключа, шифрование/дешифрование
