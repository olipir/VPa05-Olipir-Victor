# Напоминалка

Приложение-напоминалка для Windows 11 на Python с графическим интерфейсом tkinter, хранением в SQLite3 и системными уведомлениями.

## Функции

- Добавление напоминания (заголовок, описание, дата и время)
- Удаление напоминания
- Смена статуса: "Готово" / "Отменено"
- Автоматический перевод просроченных в статус "Просрочено"
- Фильтрация списка по статусу
- Встроенный календарь для выбора даты + выбор времени из списка
- Кнопка "Тест уведомления" для проверки работоспособности уведомлений
- Уведомление при наступлении срока: **win10toast**, при отсутствии — popup-окно поверх всех окон
- Фоновая проверка просроченных напоминаний каждые 30 секунд (работает при свёрнутом окне)
- Кнопки действий расположены вертикально справа от списка

## Структура проекта

```
├── main.py        # Точка входа
├── database.py    # Слой SQLite3 (Singleton, CRUD, mark_overdue)
├── gui.py         # Интерфейс tkinter (DatePicker, TimePicker, TreeView, фильтр)
├── notifier.py    # Уведомления (win10toast → popup fallback)
├── scheduler.py   # Фоновый поток проверки просроченных
├── run_app.ps1    # PowerShell-скрипт для запуска
└── reminders.db   # База данных (создаётся автоматически)
```

## Установка

```powershell
pip install win10toast
```

Зависимость опциональна: если `win10toast` не установлен, уведомления показываются в виде popup-окна.

## Запуск

```powershell
python main.py
```

Или через PowerShell-скрипт:

```powershell
.\run_app.ps1
```

## Статусы напоминаний

| Статус | Описание |
|---|---|
| Ожидает | Напоминание активно, ожидает срабатывания |
| Готово | Отмечено пользователем как выполненное |
| Просрочено | Время срабатывания прошло |
| Отменено | Отменено пользователем |
